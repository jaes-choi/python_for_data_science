## Notebook 사용할 때 편리한 Shortcut들 정리


### cell 실행
- [ctrl]-[enter]
- [shift]-[enter]
- [alt]-[enter]

### command / edit 모드 전환 
- [esc] / [enter]

### cell 추가
- a 
- b

### copy & paste, delete, undo
- c
- v
- x dd
- z

### cell 나누고 합치기
- [ctrl]-[shift]-[-]
- [shift]-[m]

### code / markdown 전환
- y / m

### save
- [ctrl]-[s]

### interrupt & restart kernel
- ii
- 00

### output 표시 선택
- o

### help
- h
- p

## Cell Edit 작업시 코드 완성
- 몇 자 입력 --> [Tab]
- 메소드들의 parameter 확인 --> [Tab][Tab]
